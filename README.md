# Simulador Tático para Avaliação de IA

Projeto de TCC focado na análise de qualidade estratégica de agentes em ambientes táticos.

## Objetivo
Criar um ambiente de simulação para comparar estratégias de tomada de decisão em IA e propor um modelo otimizado.

## Tecnologias
- Godot 2D
- Python (análise de dados)
